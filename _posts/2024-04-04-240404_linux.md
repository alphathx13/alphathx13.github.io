## protocol
- 서버와 클라이언트간의 약속
HTTP : 웹서비스
SSH(SFTP) : 리눅스 시스템 원격 제어
FTP : 원격 파일 전송

### SSH용 클라이언트
- 윈도우 : putty
- 리눅스 : ssh
- filezilla : 파일전송에 특화

## filezilla
- 설치해서 파일 전송
- 해당 파일을 site1로 이동
- site1로 접속해서 이미지 파일 확인
- site1 에 a.html을 생성하고 <img>를 이용해서 이미지 보여주도록
- 127.0.0.1:8011/a.html로 접속해서 정상적으로 작동하는지 확인
